
# Coffee Sales Data Analysis Dashboard

Thsi project focuses on analyzing a coffee sales datset downloaded from [here](https://mavenanalytics.io/data-playground?order=date_added%2Cdesc&search=coffee%20sales%20) to anlyze the sales trends, product performanc and store-wise distribution. The analysis was conducted using mirosoft excel and power query editor to clean, transform and visualize the data.

## Objective 
The primary objective of this project is to provide a clear undersatanding of customer purchasing pattern, identify top products and store locations and highlight the insights through visuals. These insights will be helpful for decision-makinf for sales , marketing and inventory management.
